# 💊 스마트 복약 도우미 서비스 - MediBot

## 👀 서비스 소개
* **서비스명** : MediBot - 당신의 건강한 복약 습관을 위한 AI 기반 복약 도우미
* **서비스 설명** : 약 이름을 자동으로 인식하고, 복약 알림, 부작용 기록, AI 챗봇 등 다양한 기능을 통해 사용자의 건강을 지켜주는 스마트 복약 관리 서비스입니다.

## 📅 프로젝트 기간
2025.03.14 ~ 2025.04.09

## ⭐ 주요 기능
* 약 이름 자동 인식 (OCR)
* 복약 스케줄 관리 및 알림
* 부작용 메모 기록
* 복약 패턴 리포트 제공
* AI 약물 상담 챗봇
* 주변 약국/병원 검색 (Kakao Map API)
